## Add Your Logo Images

To complete the logo setup, you need to add your logo files to the `/public` folder:

### Required Files:

1. **Main Logo** (for sidebar)
   - File: `huttle-logo.png`
   - Location: `/public/huttle-logo.png`
   - This is the horizontal "Huttle AI" text logo
   - Recommended size: Height ~200-400px (width will auto-adjust)

2. **Favicon** (for browser tab icon)
   - File: `favicon.png`
   - Location: `/public/favicon.png`
   - This is the circular gradient "O" icon
   - Recommended size: 512x512px or 256x256px

### How to Add:

1. Save the "Huttle AI" text logo as `huttle-logo.png`
2. Save the circular icon as `favicon.png`
3. Place both files in the `/public` folder
4. Refresh your browser - the logos will appear!
